# Life Timeline

A modern glassmorphism one-page website for Mike & Lynne Patti's life timeline.

## Features

- Responsive glassmorphism UI
- Life timeline from early life through long-term legacy
- Sections for faith, family, finance, business, and vision
- Static site ready for GitHub Pages

## Local preview

Open `index.html` directly in a browser.

## Deploy

This repo includes a GitHub Pages workflow in `.github/workflows/pages.yml`.
